Oque é logica de programação ?

Lógica de programação é o modo como se escreve um programa de computador, um algoritmo. Um algoritmo é uma 
sequência de passos para se executar uma função. Um exemplo de algoritmo, fora da computação, é uma receita de 
bolo. Na receita, devem-se seguir os passos para o bolo ficar pronto e sem nenhum problema.



O que são algoritmos e pseudocódigo 17:57

° Algoritmo é uma sequência finita de ações executáveis que visam obter uma solução para um determinado tipo de 
problema. Traduzindo, ele é uma receita que mostra passo a passo os procedimentos necessários para a 
realização de uma tarefa.

° Um pseudocódigo é um método de escrever um algoritmo
